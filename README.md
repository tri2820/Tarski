# Tarski theorem prover
![Proof of concept](https://media.giphy.com/media/uJ5Hs8yxaSO1NcbdcJ/giphy.gif)

- Online interactive first-order logic theorem prover with Tarski axiom set for Euclidean geometry.
- Built with Elm.
